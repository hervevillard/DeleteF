DeleteF — AI settings

DeleteF works without AI: with AI off (the default) it uses local heuristics only and makes zero network calls.

With AI on, DeleteF runs an agent: DeepSeek decides which conversations to delete (optionally filtered by the instruction you type in the panel), and the extension carries out each click. In this mode the agent can read the list, so real contact names and visible text are sent to DeepSeek. Use AI off if you don't want any data leaving your browser.

Stored locally in this browser only (browser.storage.local). Never sent anywhere except to DeepSeek when the AI fallback runs.
⚠️ Enabling AI sends conversation names and visible text to DeepSeek (a third-party API) so the agent can decide what to delete. Leave it Off for zero network calls and no data egress.
