DeleteF — AI fallback settings

DeleteF works without AI. If you provide a DeepSeek API key, the extension will only call DeepSeek when its built-in selectors fail to find a button, to recover automatically. It sends a redacted snapshot of the page structure (tags, roles, aria-labels) — not contact names or message text.

Stored locally in this browser only (browser.storage.local). Never sent anywhere except to DeepSeek when the AI fallback runs.
⚠️ Enabling AI sends a redacted page structure to DeepSeek (a third-party API). Leave it Off if you want the extension to make zero network calls.
